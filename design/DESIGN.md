# 短信打卡助手 — 设计规范

## 产品概述

Android App，自动监听短信/通知，匹配关键字后打开钉钉完成极速打卡。
品牌气质："安静的同事"——冷静、克制、不刷存在感，关键时刻准时出现。

## 屏幕结构

### 屏幕 1：仪表盘（`MainFragment`）

从上到下：
1. **Status Bar** — iOS 风格，9:41 时间 + 信号/WiFi/电池图标
2. **Header** — 状态圆点（绿=运行，灰=关闭）+ 日期 + 标题 + "?"帮助按钮
3. **服务开关** — 126px 圆形按钮居中，蓝色（开）/灰色（关），闪电 SVG 图标，开启时有扩散光晕动画
4. **统计条** — 三列等宽：今日触发次数 / 最后触发时间 / 运行状态。背景白色 16px 圆角，列间 1px 竖线分隔，数字 36px Bold 蓝色
5. **触发记录** — Section 标题（"触发记录"蓝色 13px）+ 白色卡片列表。每条：时间(58px宽等宽) + 分类徽章(蓝/绿/灰底) + 内容预览(截断)  
6. **空状态** — 当开关关闭时，统计区变灰（opacity 0.4, grayscale），日志区变灰并显示"服务未开启，暂无触发记录"
7. **底部导航栏** — 仪表盘 · 设置，选中蓝未选中灰，毛玻璃背景

### 屏幕 2：设置（`SettingsFragment`）

iOS 原生 inset grouped table view 布局：
1. **核心配置** section — 关键字管理（chevron → push）、白名单应用（chevron → push）
2. **时间策略** section — 上班打卡（toggle + 时间 → sheet）、下班打卡（toggle + 时间 → sheet）、触发延迟（slider 0-30s）、主动触发（toggle）
3. **系统权限** section — 短信权限 / 通知监听 / 电池优化 / 悬浮窗权限（灰色图标 + 尾部"已授权"文字 → 点击出说明弹窗）
4. **Section footer** — "所有权限均已授权，服务可正常运行。"

每行规范：16px 标题 Regular、左侧 28px 图标（蓝色或灰色）、右侧 chevron 或 toggle 或值文字、行间 1px 分割线（左边距 16px 对齐文字）。

### 屏幕 3：首次引导弹窗

居中模态，白色 24px 圆角卡片：
- 标题："欢迎使用短信打卡助手"
- 3 个步骤块（带数字圆圈）：前置条件 / 必须权限 / 使用方法
- 底部蓝色胶囊按钮 "知道了"
- 首次访问自动弹出，localStorage 记住已关闭状态

## 二级/三级页面（Navigation Stack）

### 触发关键字管理
- Push 进入，iOS 导航栏（返回 + 标题 + "添加"按钮）
- 关键字列表：白色分组行 + × 删除按钮 + 空态提示
- 点击关键字 → 底部 Sheet 编辑弹窗（输入框 + 取消/保存）
- 点击"添加"→ 底部 Sheet 添加弹窗
- 数据持久化到 localStorage

### 白名单应用管理
- Push 进入，iOS 导航栏
- 应用列表：图标(灰色圆角方块首字母) + 名称 + 状态说明 + iOS toggle 开关
- 关闭某个应用后，来自该应用的通知不触发打卡
- 设置页摘要文字实时同步

### 时间选择器
- 底部 Sheet，iOS 样式（拖拽手柄 + 标题 + 输入区 + 取消/确定）
- 开始/结束时间独立输入（时:分），数字键盘
- 点击"确定"后设置页摘要文字实时更新

### 日志详情弹窗
- 居中模态卡片
- 显示：分类徽章、时间、完整原文、元数据(来源号码/耗时/响应状态)
- 蓝色"关闭"按钮

### 权限说明弹窗
- 居中模态卡片
- 显示：权限名称、用途说明、当前状态
- "知道了"按钮

## 交互规范

| 交互 | 效果 |
|------|------|
| 开关切换 | 开启：图标弹跳动画(cubic-bezier 0.34,1.56,0.64,1) + 光晕扩散 700ms；关闭：平滑过渡 |
| Tab 切换 | 页面切换（display 切换），无动画 |
| Push 导航 | 页面从右滑入 320ms（cubic-bezier 0.25,0.1,0.25,1），返回时向左滑出 |
| Sheet 弹窗 | 从底部滑入 300ms（cubic-bezier 0.32,0.72,0,1），半透明遮罩 |
| 模态弹窗 | fade in 300ms，居中展示 |
| 列表行按压 | `:active` 时背景 3% 黑色叠加 |
| Slider 拖动 | 实时光标位置 + 延迟秒数同步更新 |

## 组件规格

### 服务开关圆钮
- 126×126px 圆形
- On: `background: #007AFF`, `box-shadow: 0 8px 32px rgba(0,122,255,0.30)`
- Off: `background: #C7C7CC`, no shadow
- 点击缩放 0.94 + 弹跳动画

### iOS Toggle
- 51×31px 胶囊
- On: `background: #34C759`
- Off: `background: #E5E5EA`
- 白色圆钮 27px，投影 `0 2px 4px rgba(0,0,0,0.15)`

### 统计条
- 白色背景 16px 圆角，1px `#E5E5EA` 边框
- 内部三列 grid，列间 1px 竖线（高 36px）
- 数字 36px Bold `#007AFF`
- 标签 12px Regular `#8E8E93`

### 导航栏（子页面）
- 高度 44px，毛玻璃背景
- 返回按钮：蓝色 chevron + 父页面名
- 标题居中 16px SemiBold
- 操作按钮右侧（蓝色 16px）

## 禁止项

- ❌ 渐变背景
- ❌ 装饰性图标/emoji
- ❌ 无意义动画
- ❌ 全大写英文按钮
- ❌ 大于 1px 的单侧彩色边框
- ❌ 多个强调色同时出现
- ❌ 手风琴式折叠面板（设置页已改为 iOS 原生 grouped table）
- ❌ 灰色文字在灰色背景上（确保 WCAG AA 对比度）
